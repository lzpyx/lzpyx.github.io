环境搭建

Posted on 2022-04-21 In 工具

环境安装

查看&删除已有环境

  • 查看现有的环境conda info —envs

  • 删除环境conda remove -n yolov5 —all

使用指定源安装

从0开始搭建

  • 安装anaconda

  • 安装cuda 10.2/cuda 11.0

  • 创建虚拟环境

    conda create -n yolov5 python=3.8

    conda activate yolov5

  • 更换源

    conda config —add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/

    conda config —add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/

    conda config —add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/pytorch/

  • 安装pytorch和torchvision对应版本

    cuda10.2:

    conda install pytorch== 1.7.1 torchvision==0.8.2 cudatoolkit=10.2

    cuda11.0:

    conda install pytorch== 1.7.1 torchvision== 0.8.2 torchaudio==0.7.2 cudatoolkit=11.0
