博客搭建

参考视频

安装node.js

直接从官网下载长期维护版或者release版本

下载16.14.2的情况下可能和hexo不适配，参考hexo deploy上传github全是空白文件问题解决

下载release版本，12.17.0

node -v检查是否安装成功

github创建github pages

新建一个repository，名字为username.github.io (example:lzpyx.github.io)

选择public属性

gitforwin安装

cmd输入git验证安装是否成功

git config –global user.email”1161682912@qq.com“

git config –global user.name”lzpyx”

github ssh配置

打开Git安装目录下的git-bash.exe

输入 (example: ssh-keygen -t rsa -C “1161682912@qq.com“)

注意返回的括号中的/c/Users/lzpyx/.ssh/id_rsa，后面会用到这个文件夹中的id_rsa.pub文件中的内容

连续回车，相当于不设置密码

打开github，在setting中选择SSH，新建，把id_rsa.pub文件中的内容复制到key，title就直接username（lzpyx）即可

hexo配置

在D盘新建blog文件夹

cmd切换到blog文件夹，然后开始安装

hexo官网

直接复制指令到cmd进行安装

创建

选择中意的主题/模板

next

下载zip，然后解压，把文件夹改名为next，放入theme，修改_config.yml中的theme为next

安装依赖

安装部署工具（方便以后更新）
npm install hexo-deployer-git -sav
