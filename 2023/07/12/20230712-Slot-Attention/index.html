Local Slot Attention for Vision-and-Language Navigation

Motivation

еҪ“еүҚзҡ„еҹәдәҺtransformerзҡ„и§ЈеҶіVLNзҡ„жЁЎеһӢдёӯеӯҳеңЁдёӨдёӘй—®йўҳпјҡ

  • жЁЎеһӢзӢ¬з«Ӣең°еӨ„зҗҶжҜҸдёӘи§ҶеӣҫпјҢиҖҢдёҚж•ҙдҪ“зҡ„иҖғиҷ‘
  • self-attentionдёӯжІЎжңүеҜ№з©әй—ҙиҝӣиЎҢйҷҗеҲ¶пјҢеҸҜиғҪеј•е…ҘйўқеӨ–еҷӘеЈ°пјҢеңЁеӨ§иҢғеӣҙеҶ…иҝӣиЎҢself-attentionзҡ„и®Ўз®—д»Јд»·иҫғеӨ§

PREVALENTдёӯдёҖдёӘи§ҶзӮ№зҡ„жүҖжңү36дёӘи§ҶеӣҫеңЁжҜҸдёӘж—¶й—ҙжӯҘеҪјжӯӨе…іжіЁпјҢд»Ҙж•ҙеҗҲжқҘиҮӘж•ҙдёӘеңәжҷҜзҡ„дҝЎжҒҜгҖӮиҝҷдёҚд»…еңЁи®Ўз®—дёҠжҳҜжҳӮиҙөзҡ„пјҢиҖҢдё”иҝқиғҢеёёиҜҶпјҢеӣ дёәе…·жңүеӨ§з©әй—ҙи·қзҰ»зҡ„еҢәеҹҹйҖҡеёёжҳҜдёҚзӣёе…ізҡ„гҖӮжқҘиҮӘиҝңзЁӢи§Ҷеӣҫзҡ„иҝҷз§ҚдҝЎжҒҜеҸҜиғҪжҲҗдёәе№Іжү°еҜјиҲӘеҶізӯ–зҡ„еҷӘеЈ°гҖӮ

еңЁRecurrent VLN-BertдёӯпјҢд»…е…іжіЁеҖҷйҖүи§ҶеӣҫгҖӮиҝҷеҮҸе°‘дәҶеҶ…еӯҳзҡ„дҪҝз”ЁпјҢдҪҶжҳҜд№ҹе°Ҷagentзҡ„ж„ҹеҸ—йҮҺйҷҗеҲ¶еңЁдәҶеӯӨз«Ӣзҡ„candidate viewдёӯпјҢи§Ҷи§үзәҝзҙўеҸҜиғҪеңЁcandidate viewдёӯиў«иЈҒеүӘжҲ–жЁЎзіҠпјҢе№¶дё”дёҚиғҪеҜ№еә”дәҺиҜӯиЁҖжҢҮд»ӨгҖӮ

Methodology

The state of navigation is composed of the history of visual perceptions and the embedding of language instructions.

The state of the agent includes its pose, which is the embedding of its current heading and elevation

Panoramic Action Space

  • е…ЁжҷҜеҠЁдҪңз©әй—ҙпјҡжҜҸдёӘviewpointиў«иЎЁзӨәдёәз”ұ3 Г— 12viewsз»„жҲҗзҡ„е…ЁжҷҜеӣҫпјҢжҜҸдёӘи§ҶеӣҫеҜ№еә”дәҺзү№е®ҡзҡ„иҲӘеҗ‘и§’е’ҢдҝҜд»°и§’пјҢFOV(и§Ҷи§’иҢғеӣҙ)дёә60еәҰпјҢеҲҶиҫЁзҺҮдёә$640\times 480$пјҢиҲӘеҗ‘и§’иҢғеӣҙдёә0еҲ°360еәҰпјҢй—ҙйҡ”дёә30еәҰпјҢдҝҜд»°и§’дёә0гҖҒ30жҲ–-30еәҰгҖӮ

  • candidate viewпјҡеңЁ36дёӘviewsдёӯпјҢжңүдәӣviewжҳҜжңқеҗ‘еҸҰеӨ–зҡ„viewpointзҡ„пјҢиў«з§°дёәcandidate view

  • жң¬ж–ҮеҲ©з”ЁResNet-152 model pre-trained on the Place365 datasetжқҘеҜ№viewйў„жҸҗеҸ–зү№еҫҒпјҢжҜҸдёӘviewиў«зј–з Ғдёә2048з»ҙзҡ„tensor
  • дёәдәҶиЎЁзӨәжҜҸдёӘviewзҡ„дҪҚзҪ®зү№еҫҒпјҢйҮҮз”ЁйҮҚеӨҚ32ж¬Ўзҡ„$(sin(рқң“рқ‘–),cos(рқң“рқ‘–),sin(рқң”рқ‘–),cos(рқң”рқ‘–))$зҡ„еӣӣе…ғж•°дҪңдёәз¬¬iдёӘеҖҷйҖүи§Ҷеӣҫзҡ„и§’еәҰзү№еҫҒпјҢе…¶дёӯрқң“рқ‘–пјҢрқң”рқ‘–еҲҶеҲ«иЎЁзӨәе…¶зӣёеҜ№дәҺagentзҡ„еҪ“еүҚжңқеҗ‘зҡ„иҲӘеҗ‘и§’е’ҢдҝҜд»°и§’пјҢиҝҷдёҖи§’еәҰзү№еҫҒзҡ„з»ҙеәҰдёә128пјҢи§’еәҰзү№еҫҒе°Ҷе’ҢеӣҫеғҸзү№еҫҒиҝӣиЎҢжӢјжҺҘпјҢз»ҙеәҰдёә2176пјҢдҪңдёәview featureпјҢжңҖеҗҺеҫ—еҲ°еұһдәҺдёҖдёӘviewpointзҡ„feature map shapeдёә(36, 2176)

Revisiting Recurrent VLN-Bert

  • йҖҡиҝҮBERTеҜ№language instructionsиҝӣиЎҢзј–з ҒпјҢе№¶жҠҠ[CLS] tokenпјҲ$L_{cls}$пјүдҪңдёәinitial navigation state, [CLS] tokenдјҡеңЁеҜјиҲӘиҝҮзЁӢдёӯдёҚж–ӯжӣҙж–°пјҢе…¶д»–tokenпјҲиЎЁзӨәдёә$L_{/cls}$пјүеӣәе®ҡдёҚеҸҳ
  • еңЁaction selectionйҳ¶ж®өпјҢmulti-layer cross-attention moduleдёӯе°Ҷview featuresе’Ңstate embedding ([CLS] token)жӢјжҺҘиө·жқҘдҪңдёәqueries, instruction embeddingзҡ„еӣәе®ҡtoken($L_{/cls}$)дҪңдёәcontext(keys and values), з„¶еҗҺи®Ўз®—cross-attention

$\mathrm{L}_{c l s}^{X}, \mathrm{~V}_{t}^{X}=\text { Attention }_{\text {cross }}\left(\left[\mathrm{L}_{c l s} ; \mathrm{V}_{t}\right], \mathrm{L}_{/ c l s}\right)$

cross-attention moduleзҡ„иҫ“еҮәиў«еҲҶдёәдёӨйғЁеҲҶдҪңдёә[$\mathrm{L}_{c l s}^{X}$; $\mathrm{~V}_{t}^{X}$]гҖӮ$\mathrm{L}_{c l s}^{X}$иЎЁзӨә[CLS]tokenзҡ„embeddingпјҲеҚіstate embeddingпјүпјҢе№¶дё”$\mathrm{~V}_{t}^{X}$иЎЁзӨәи§Ҷи§үиҫ“е…Ҙзҡ„embeddingгҖӮ

  • multi-layer self-attention moduleж•ҙеҗҲcandidate viewsзҡ„дҝЎжҒҜеҲ°[CLS] tokenдёӯпјҢжңҖеҗҺдёҖеұӮself-attention layerзҡ„[CLS] tokenеҜ№жүҖжңүcandidate viewsзҡ„attention weightsе°ұжҳҜaction selection scoreпјҢagentйҖүжӢ©еҫ—еҲҶжңҖй«ҳзҡ„candidate viewз§»еҠЁжҲ–еҒңжӯў
  • з„¶еҗҺи®Ўз®—language embedding$\mathrm{L}_{/c l s}^{X}$е’Ңvisual embeddingзӮ№д№ҳиҺ·еҫ—еӨҡжЁЎжҖҒзү№еҫҒпјҢеҶҚз»ҸиҝҮlinear projectionеҫ—еҲ°дёӢдёҖдёӘж—¶й—ҙжӯҘзҡ„navigation stateпјҲеҚіз”ЁиҝҷдёҖstateжӣҝд»Ј$L_{cls}$пјҢе®ҢжҲҗзҠ¶жҖҒжӣҙж–°пјү

Local Attention Mask

еҰӮжһңslotпјҲcandidate viewпјүжӣҙйқ иҝ‘дёҠдёӢж–Үи§ҶеӣҫпјҢеёҢжңӣжіЁж„ҸеҠӣжқғйҮҚжӣҙй«ҳпјҢеҰӮжһңslotе’ҢдёҠдёӢж–Үи§Ҷеӣҫи·қзҰ»иҫғиҝңпјҢеҲҷеёҢжңӣжіЁж„ҸеҠӣжқғйҮҚжӣҙдҪҺгҖӮеӣ жӯӨдҪңиҖ…йҮҮз”ЁдәҶдёҖдёӘз®ҖеҚ•зҡ„local attention maskжқҘйҷҗеҲ¶жіЁж„ҸеҠӣзҡ„иҢғеӣҙпјҢдҪҝslot attentionжӣҙзЁіе®ҡгҖӮ

  • 36дёӘviewsиў«и§ҶдҪң$3\times 12$зҡ„зҪ‘ж ј
  • дёҫдҫӢжқҘиҜҙпјҢйҖҡиҝҮеә”з”Ё$3\times 3$еұҖйғЁжҺ©жЁЎпјҢcandidate viewд»…е…іжіЁе‘Ёеӣҙзҡ„$3\times 3$еҢәеҹҹгҖӮ

Experiment

Datasets

The R2R dataset contains 10,567 panoramic views of 90 real-world in-door environments and 7,189 trajectories, each one is attached with three instructions written in natural language

Metrics

  • Navigation Error(NE), the shortest path length between the agentвҖҷs final position and the goal location.
  • Success Rate (SR), the percentage of the agent successfully stops within 3 meters of the goal location.
  • Oracle Success Rate (OSR), as long as the agent is ever within 3 meters ofthe goal location in its trajectory, it is considered successful.
  • Success rate weighted by Path Length (SPL), trades-off success rate against trajectory length, higher SPL means more efficient navigation and less exploration.
