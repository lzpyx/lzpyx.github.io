LOSTеңЁж— зӣ‘зқЈеҚ•зӣ®ж ҮеҸ‘зҺ°пјҲе·ҰпјүпјҢеӨҡзӣ®ж ҮеҸ‘зҺ°пјҲдёӯпјүпјҢзӣ®ж ҮжЈҖжөӢпјҲеҸіпјүдёӯзҡ„еә”з”ЁпјҢеңЁзӣ®ж ҮжЈҖжөӢеә”з”ЁдёӯпјҢLOSTеҸ‘зҺ°зҡ„еҜ№иұЎе°Ҷиў«иҒҡзұ»еҲ°зұ»еҲ«дёӯпјҢз„¶еҗҺдҪҝз”ЁжҚ®зұ»ж ҮзӯҫжқҘи®ӯз»ғз»Ҹе…ёзҡ„еҜ№иұЎжЈҖжөӢеҷЁ

Introduction

йҖҡеёёдҪҝз”ЁеҢәеҹҹе»әи®®дҪңдёәиҫ“е…ҘпјҢж— зӣ‘зқЈеҜ№иұЎеҸ‘зҺ°еҲ©з”ЁжқҘиҮӘж•ҙдёӘеӣҫеғҸйӣҶеҗҲзҡ„дҝЎжҒҜе№¶жҺўзҙўеӣҫеғҸй—ҙзӣёдјјжҖ§д»Ҙж— зӣ‘зқЈж–№ејҸе®ҡдҪҚеҜ№иұЎ

Our localization method stays at the level of a single image, rather than exploring inter-image similarity, which makes it linear w.r.t. the number of images and thus highly scalable.

жҲ‘д»¬зҡ„е®ҡдҪҚж–№жі•еҒңз•ҷеңЁеҚ•дёӘеӣҫеғҸзҡ„зә§еҲ«пјҢиҖҢдёҚжҳҜжҺўзҙўеӣҫеғҸй—ҙзҡ„зӣёдјјжҖ§пјҢиҝҷдҪҝе…¶жҲҗдёәзәҝжҖ§ w.r.tгҖӮеӣҫеғҸзҡ„ж•°йҮҸпјҢеӣ жӯӨе…·жңүй«ҳеәҰеҸҜжү©еұ•жҖ§гҖӮдҫқиө–дәҺиҮӘзӣ‘зқЈtransformerзҡ„ејәеӨ§еҠҹиғҪпјҢе№¶дё”еҸӘиҖғиҷ‘еӣҫеғҸеҶ…зҡ„зӣёдјјжҖ§пјҢжң¬ж–№жі•еҸҜд»Ҙз”ЁеҫҲе°‘зҡ„и®Ўз®—еңЁеҚ•дёӘеӣҫеғҸдёӯе®ҡдҪҚеҜ№иұЎгҖӮ

дё»иҰҒе·ҘдҪңпјҡ

  • еҲ©з”ЁиҮӘзӣ‘зқЈйў„и®ӯз»ғtransformerжҸҗеҸ–зү№еҫҒпјҢдҪҝз”ЁеӣҫеғҸдёӯиЎҘдёҒзҡ„зӣёе…іжҖ§жҸҗеҮәдёҖз§ҚеҜ№дәҺдёҖеј еӣҫзүҮзҡ„еҚ•зӣ®ж Үе®ҡдҪҚж–№жі•пјҢиҝҷдёҖж–№жі•е…ідәҺж•°жҚ®йӣҶеӨ§е°Ҹе…·жңүзәҝжҖ§еӨҚжқӮеәҰгҖӮ
  • еҲ©з”Ёе®ғжқҘи®ӯз»ғдёҺзұ»ж— е…іе’Ңзұ»ж„ҹзҹҘзҡ„ж— зӣ‘зқЈеҜ№иұЎжЈҖжөӢеҷЁпјҢиғҪеӨҹеҮҶзЎ®ең°е®ҡдҪҚжҜҸдёӘеӣҫеғҸзҡ„еӨҡдёӘеҜ№иұЎпјҢе№¶дё”еңЁзұ»ж„ҹзҹҘзҡ„жғ…еҶөдёӢпјҢе°Ҷе®ғд»¬еҲҶз»„еҲ°иҜӯд№үдёҖиҮҙзҡ„зұ»дёӯ
  • еңЁж— зӣ‘зқЈеҜ№иұЎеҸ‘зҺ°ж–№йқўд»ҘжҳҫзқҖдјҳеҠҝи¶…и¶ҠдәҶжңҖе…Ҳиҝӣзҡ„жҠҖжңҜ

Methodology

жҰӮиҝ°

е°Ҷж„ҹе…ҙи¶Јзҡ„еӣҫеғҸеҲҶжҲҗеӨ§е°Ҹзӣёзӯүзҡ„еқ—е№¶е°Ҷе…¶жҸҗдҫӣз»ҷ DINO жЁЎеһӢгҖӮжң¬ж–ҮдҪҝз”ЁдәҶжңҖеҗҺдёҖдёӘжіЁж„ҸеҠӣеұӮзҡ„key componentжқҘи®Ўз®—дёҚеҗҢpatchд№Ӣй—ҙзҡ„зӣёдјјжҖ§пјҢиҖҢдёҚжҳҜcls tokenгҖӮиҝҷж ·еҒҡзҡ„зӣ®зҡ„жҳҜдҪҝеҫ—еҸҜд»ҘйҖҡиҝҮйҖүжӢ©иҫғе°‘зҡ„зӣёдјјpatchesжқҘе®ҡдҪҚзӣ®ж Үзҡ„дёҖйғЁеҲҶпјҢз§°дёәseedгҖӮseedйҖүеҸ–зҡ„ж ҮеҮҶжҳҜеҹәдәҺз»ҸйӘҢи§ӮеҜҹ(empirical observation)пјҡеүҚжҷҜзӣ®ж ҮеҶ…зҡ„patchesеҪјжӯӨд№Ӣй—ҙзҡ„зӣёе…іжҖ§й«ҳдәҺдёҺиғҢжҷҜpatchesд№Ӣй—ҙзҡ„зӣёе…іжҖ§гҖӮжҺҘзқҖеҗ‘еҲқе§Ӣзҡ„seedж·»еҠ дёҺд№Ӣй«ҳеәҰзӣёе…ізҡ„е…¶д»–patchesпјҢз§°дёәseed expansionгҖӮжңҖеҗҺйҖҡиҝҮи®Ўз®—жҜҸдёӘimage patchе’ҢжүҖйҖүзҡ„seed patchesзҡ„зӣёдјјжҖ§жқҘжһ„йҖ дёҖдёӘдәҢеҖјеҲҶеүІжҺ©з ҒпјҢе№¶е°ҶеҜ№иұЎзҡ„иҫ№з•ҢжЎҶжҺЁж–ӯдёәзҙ§еҜҶеҢ…еӣҙиҜҘжҺ©з ҒдёӯеҢ…еҗ«initial seedзҡ„жңҖеӨ§иҝһйҖҡеҲҶйҮҸзҡ„жЎҶгҖӮ

дҪҝз”Ёе®ҡдҪҚжЎҶдҪңдёәи®ӯз»ғж ҮзӯҫжқҘи®ӯз»ғдёҖдёӘдёҺзұ»еҲ«ж— е…ізҡ„еҜ№иұЎжЈҖжөӢеҷЁпјҢиғҪеӨҹеҫ—еҲ°дёҖдёӘжӣҙеҮҶзЎ®зҡ„еҜ№иұЎе®ҡдҪҚжЁЎеһӢпјҢиғҪеӨҹжЈҖжөӢеӣҫеғҸдёӯзҡ„еӨҡдёӘеҜ№иұЎгҖӮдҪңиҖ…е°ҶжӯӨд»»еҠЎз§°дёәж— зӣ‘зқЈзҡ„дёҺзұ»еҲ«ж— е…ізҡ„еҜ№иұЎжЈҖжөӢunsupervised class-agnostic object detection пјҲе°Ҫз®Ўиў«з§°дёәж— зӣ‘зқЈпјҢдҪҶе®ғеҸҜиғҪдјҡжұӮеҠ©дәҺиҮӘзӣ‘зқЈпјүгҖӮ

йҖҡиҝҮдҪҝз”ЁиҒҡзұ»жҠҖжңҜе°ҶеұҖйғЁеҜ№иұЎеҲҶз»„еҲ°и§Ҷи§үдёҖиҮҙзҡ„зұ»еҲ«дёӯпјҢиғҪеӨҹеңЁжІЎжңүд»»дҪ•дәәе·Ҙзӣ‘зқЈзҡ„жғ…еҶөдёӢпјҢдҪҝз”Ёйў„жөӢзҡ„еҜ№иұЎдҪҚзҪ®еҸҠе…¶йӣҶзҫӨ ID дҪңдёәзңҹе®һж ҮжіЁи®ӯз»ғе…·жңүзұ»еҲ«ж„ҸиҜҶзҡ„еҜ№иұЎжЈҖжөӢеҷЁгҖӮжҲ‘д»¬е°ҶжӯӨд»»еҠЎз§°дёәж— зӣ‘зқЈпјҲзұ»еҲ«ж„ҹзҹҘпјүеҜ№иұЎжЈҖжөӢunsupervised (class-aware) object detectionгҖӮ

LOST

дҪңиҖ…еҒҮи®ҫеӣҫеғҸдёӯиҮіе°‘жңүдёҖдёӘеҜ№иұЎпјҢе№¶дё” LOST е°қиҜ•еңЁз»ҷе®ҡиҫ“е…Ҙзү№еҫҒзҡ„жғ…еҶөдёӢе®ҡдҪҚе…¶дёӯдёҖдёӘеҜ№иұЎгҖӮдёәжӯӨпјҢе®ғдҫқиө–дәҺйҖүжӢ©еҸҜиғҪеұһдәҺжҹҗдёӘеҜ№иұЎзҡ„patchesгҖӮжҲ‘д»¬з§°иҝҷдәӣpatchesдёәвҖңseedsвҖқгҖӮ

Initial seed selection

з§Қеӯҗзҡ„йҖүжӢ©зӯ–з•ҘеҹәдәҺд»ҘдёӢеҒҮи®ҫпјҡ

  • еүҚжҷҜзӣ®ж ҮеҶ…зҡ„patchesеҪјжӯӨд№Ӣй—ҙзҡ„зӣёе…іжҖ§й«ҳдәҺдёҺиғҢжҷҜpatchesд№Ӣй—ҙзҡ„зӣёе…іжҖ§
    • з»ҸйӘҢи§ӮеҜҹеҲ°пјҢжҠҠtransformer featureдҪңдёәpatchзҡ„representationж»Ўи¶іиҝҷдёҖеҒҮи®ҫпјҢеҜ№иұЎдёӯзҡ„patchesеҪјжӯӨжӯЈзӣёе…іпјҢдҪҶдёҺиғҢжҷҜдёӯзҡ„patchesиҙҹзӣёе…і
  • дёҖдёӘеҚ•зӢ¬еҜ№иұЎиҰҶзӣ–зҡ„еҢәеҹҹе°‘дәҺиғҢжҷҜпјҢеӣ жӯӨеӣҫеғҸдёӯзӣёе…іжҖ§еҫҲе°Ҹзҡ„patchеұһдәҺеҜ№иұЎзҡ„жҰӮзҺҮжӣҙй«ҳ
    • еҹәдәҺиҝҷдёҖеҒҮи®ҫпјҢдҪңиҖ…йҖҡиҝҮйҖүжӢ©дёҺе…¶д»–patchesжӯЈзӣёе…іж•°йҮҸжңҖе°‘зҡ„patchдҪңдёәз§Қеӯҗseed $p^*$

еҜ№дәҺдёҖеј еӣҫеғҸпјҢйҰ–е…Ҳе»әз«ӢдёҖдёӘжҸҸиҝ°patchй—ҙзҡ„зӣёдјјеәҰзҡ„graphпјҢз”ЁдәҢе…ғеҜ№з§°йӮ»жҺҘзҹ©йҳөиЎЁзӨә

д№ҹе°ұжҳҜиҜҙпјҢдёӨдёӘиҠӮзӮ№$p,q$зҡ„зү№еҫҒ $ f_p,f_q$жӯЈзӣёе…іпјҢеҲҷйҖҡиҝҮж— еҗ‘иҫ№иҝһжҺҘгҖӮз„¶еҗҺе°ҶеәҰ(degree)жңҖе°Ҹзҡ„patchдҪңдёәеҲқе§Ӣseed $p^*$

еӣҫдёӯз¬¬дәҢиЎҢзҡ„зәўиүІзӮ№е°ұжҳҜеңЁеӣҫеғҸдёӯйҖүеҸ–зҡ„еҲқе§Ӣseed $p^$пјҢзҒ°иүІд»ЈиЎЁе’Ң $ p^$зӣёдјјзҡ„patchesпјҢд№ҹе°ұжҳҜ$f_p^Tf_q\geqеҚіa_{pq}=1$гҖӮжңҖеҗҺдёҖиЎҢиЎЁзӨәзҡ„жҳҜжҜҸдёӘpatchзҡ„еҸҚеәҰеӣҫ(inverse degrees)$ 1/d_p$пјҢд»Һй»„иүІеҲ°и“қиүІд»ЈиЎЁд»Һdegreeд»ҺдҪҺеҲ°й«ҳпјҢз§ҚеӯҗзӮ№е°ұжҳҜdegreeжңҖдҪҺзҡ„зӮ№гҖӮеҸҜд»ҘеҸ‘зҺ°degreeжңҖдҪҺзҡ„patchжңҖжңүеҸҜиғҪиҗҪе…Ҙзү©дҪ“дёӯпјҢдёҺ $ p^$жӯЈзӣёе…ізҡ„е°‘ж•°еқ—д№ҹеҸҜиғҪеұһдәҺеҗҢдёҖдёӘеҜ№иұЎгҖӮ

Seed expansion

дҪңиҖ…дҫқйқ з»ҸйӘҢи§ӮеҜҹжқҘе®һзҺ°иҝҷдёҖжӯҘйӘӨпјҢеҚіеҜ№иұЎеҶ…зҡ„еғҸзҙ еҫҖеҫҖе‘ҲжӯЈзӣёе…іе№¶дё”еңЁgraphдёӯе…·жңүиҫғе°Ҹзҡ„еәҰж•°гҖӮйҖүеҸ–дёҺз§ҚеӯҗзӮ№жӯЈзӣёе…ідё”еәҰж•°жңҖдҪҺзҡ„kдёӘpatchesпјҢkзҡ„е…ёеһӢеҖјжҳҜ100

Box extraction

е°ҶеӣҫеғҸжүҖжңүpatchе’Ңз§ҚеӯҗеҢәеҹҹSзҡ„зү№еҫҒжҜ”иҫғпјҢи®Ўз®—жҺ©з Ғ

д№ҹе°ұжҳҜиҜҙе№іеқҮиҖҢиЁҖпјҢеҰӮжһңtoken q зҡ„зү№еҫҒ fq дёҺ S дёӯзҡ„tokenзҡ„зү№еҫҒжӯЈзӣёе…іпјҢеҲҷtoken q иў«и§ҶдёәеҜ№иұЎзҡ„дёҖйғЁеҲҶгҖӮдёәдәҶеҺ»йҷӨжңҖеҗҺдёҖдёӘиҷҡеҒҮзҡ„зӣёе…іиЎҘдёҒпјҢжҲ‘д»¬жңҖз»ҲйҖүжӢ© m дёӯзҡ„иҝһйҖҡеҲҶйҮҸеҢ…еҗ«еҲқе§Ӣз§ҚеӯҗпјҢе№¶дҪҝз”ЁиҜҘз»„д»¶зҡ„иҫ№з•ҢжЎҶдҪңдёәжЈҖжөӢеҲ°зҡ„еҜ№иұЎгҖӮеӣҫ 3 жҸҗдҫӣдәҶз§Қеӯҗжү©еұ•еүҚеҗҺжЈҖжөӢеҲ°зҡ„жЎҶзҡ„еӣҫзӨәгҖӮ

TPAMI2021 Affinity Attention Graph Neural Network

Affinity Attention Graph Neural Network for Weakly Supervised Semantic Segmentation дәІе’ҢеҠӣдёҺеӣҫеҚ·з§ҜзҘһз»ҸзҪ‘з»ңзҡ„з»“еҗҲ

е°Ҷејұзӣ‘зқЈиҪ¬жҚўдёәsemi-supervisedпјҲйғЁеҲҶзӮ№жҳҜжңүж ҮжіЁзҡ„пјү

Introduction

еӣҫзҘһз»ҸзҪ‘з»ңGNNеҸҜд»ҘеҸҜд»ҘзӣҙжҺҘеңЁдёҚеҗҢиҠӮзӮ№д№Ӣй—ҙе»әз«Ӣй•ҝи·қзҰ»е…ізі»пјҲиҫ№пјүпјҢиҒҡеҗҲеӨҡдёӘиҝһжҺҘиҠӮзӮ№зҡ„дҝЎжҒҜпјҢд»ҺиҖҢжҠ‘еҲ¶ж ҮзӯҫеҷӘеЈ°зҡ„иҙҹйқўеҪұе“ҚпјҢеңЁж Үзӯҫжңүйҷҗзҡ„еҚҠзӣ‘зқЈд»»еҠЎдёӯGNNд№ҹиЎЁзҺ°иүҜеҘҪгҖӮ

GraphNetзҡ„зјәзӮ№пјҡ

WSSSзӣ®еүҚзҡ„ж–№жі•дё»иҰҒеҹәдәҺеҚ·з§ҜзҘһз»ҸзҪ‘з»ңпјҢж— жі•жӯЈзЎ®жҺўзҙўе…ЁеұҖдҝЎжҒҜпјҢеӣ жӯӨйҖҡеёёдјҡеҜјиҮҙеҜ№иұЎеҢәеҹҹдёҚе®Ңж•ҙгҖӮеңЁжң¬ж–ҮдёӯпјҢдёәдәҶи§ЈеҶідёҠиҝ°й—®йўҳеј•е…ҘдәҶTransformerпјҢе®ғиҮӘз„¶ең°ж•ҙеҗҲдәҶе…ЁеұҖдҝЎжҒҜпјҢдёәз«ҜеҲ°з«Ҝзҡ„ WSSS з”ҹжҲҗжӣҙе®Ңж•ҙзҡ„еҲқе§ӢдјӘж ҮзӯҫгҖӮеҸ—Transformersдёӯзҡ„self-attentionе’ҢиҜӯд№үдәІе’ҢеҠӣд№Ӣй—ҙзҡ„еҶ…еңЁдёҖиҮҙжҖ§зҡ„еҗҜеҸ‘пјҢдҪңиҖ…жҸҗеҮәдәҶAffinity from Attention (AFA)жЁЎеқ—жқҘд»ҺTransformersдёӯзҡ„еӨҡеӨҙиҮӘжіЁж„ҸеҠӣ(MHSA)дёӯеӯҰд№ иҜӯд№үдәІе’ҢеҠӣгҖӮз„¶еҗҺеҲ©з”ЁеӯҰд№ еҲ°зҡ„дәІе’ҢеҠӣжқҘз»ҶеҢ–еҲқе§ӢдјӘж Үзӯҫд»ҘиҝӣиЎҢеҲҶеүІгҖӮжӯӨеӨ–пјҢдёәдәҶжңүж•Ҳең°еҜјеҮәеҸҜйқ зҡ„дәІе’Ңж ҮзӯҫжқҘзӣ‘зқЈ AFA е№¶зЎ®дҝқдјӘж Үзӯҫзҡ„еұҖйғЁдёҖиҮҙжҖ§local consistencyпјҢдҪңиҖ…и®ҫи®ЎдәҶдёҖдёӘеғҸзҙ иҮӘйҖӮеә”з»ҶеҢ–жЁЎеқ—пјҢиҜҘжЁЎеқ—з»“еҗҲдәҶдҪҺзә§еӣҫеғҸеӨ–и§ӮдҝЎжҒҜжқҘз»ҶеҢ–дјӘж ҮзӯҫгҖӮ

WSSSж–№жі•йҖҡеёёйҮҮз”ЁеӨҡйҳ¶ж®өжЎҶжһ¶пјҢд№ҹе°ұжҳҜе…Ҳи®ӯз»ғдёҖдёӘеҲҶзұ»жЁЎеһӢпјҢз„¶еҗҺз”ҹжҲҗзұ»жҝҖжҙ»еӣҫCAMдҪңдёәдјӘж ҮзӯҫгҖӮз»ҶеҢ–еҗҺпјҢдјӘж Үзӯҫиў«з”ЁжқҘи®ӯз»ғдёҖдёӘзӢ¬з«Ӣзҡ„иҜӯд№үеҲҶеүІзҪ‘з»ңдҪңдёәжңҖз»ҲжЁЎеһӢгҖӮ

еӨҡйҳ¶ж®өи®ӯз»ғжөҒзЁӢеӨҚжқӮзј“ж…ўпјҢдёҚеӨҹй«ҳж•ҲгҖӮзҺ°жңүзҡ„дёҖдәӣз«ҜеҲ°з«Ҝзҡ„и§ЈеҶіж–№жЎҲеҹәжң¬жҳҜеҹәдәҺCNNзҡ„пјҢCNNеҜ№е…ЁеұҖзү№еҫҒзҡ„иҺ·еҸ–иғҪеҠӣиҫғе·®гҖӮ

йҰ–е…ҲдҪҝз”ЁTransformerз”ҹжҲҗCAMдҪңдёәеҲқе§ӢдјӘж ҮзӯҫпјҢд»ҘйҒҝе…ҚCNNзҡ„еҶ…еңЁзјәйҷ·пјҢз”ұдәҺ MHSA дёӯзҡ„иҜӯд№үдәІе’ҢжҖ§еҫҲзІ—зіҷпјҢдҪңиҖ…жҸҗеҮәAffinity from Attention (AFA) жЁЎеқ—пјҢж—ЁеңЁеҜјеҮәеҸҜйқ зҡ„дјӘдәІе’ҢеҠӣж ҮзӯҫжқҘзӣ‘зқЈд»ҺTransformerдёӯзҡ„ MHSA еӯҰд№ зҡ„иҜӯд№үдәІе’ҢжҖ§гҖӮз„¶еҗҺдҪҝз”ЁеӯҰд№ еҲ°зҡ„дәІе’ҢеҠӣйҖҡиҝҮйҡҸжңәжёёиө°дј ж’ӯжқҘдҝ®ж”№еҲқе§ӢдјӘж ҮзӯҫпјҢиҝҷеҸҜд»Ҙжү©ж•ЈеҜ№иұЎеҢәеҹҹе№¶жҠ‘еҲ¶й”ҷиҜҜжҝҖжҙ»зҡ„еҢәеҹҹгҖӮдёәдәҶдҪҝAFA еҜјеҮәй«ҳеәҰеҸҜдҝЎзҡ„дјӘдәІе’Ңж Үзӯҫе№¶зЎ®дҝқдј ж’ӯзҡ„дјӘж Үзӯҫзҡ„еұҖйғЁдёҖиҮҙжҖ§пјҢжҲ‘д»¬иҝӣдёҖжӯҘжҸҗеҮәдәҶдёҖдёӘеғҸзҙ иҮӘйҖӮеә”з»ҶеҢ–жЁЎеқ— (PAR)гҖӮPAR жңүж•Ҳең°ж•ҙеҗҲдәҶеұҖйғЁеғҸзҙ зҡ„ RGB е’ҢдҪҚзҪ®дҝЎжҒҜд»Ҙз»ҶеҢ–дјӘж ҮзӯҫпјҢд»ҺиҖҢжӣҙеҘҪең°дёҺдҪҺзә§еӣҫеғҸеӨ–и§ӮеҜ№йҪҗгҖӮ
